# 基于同态加密的网恋匹配系统

## Usage
1. `pipenv` (recommend)
    1. `pipenv install -r ./requirements.txt`
    2. `pipenv run python ./main.py`

2. `pip` 
    1. `pip install -r requirements.txt`
    2. `python ./main.py`

3. Compile
    1. Run `pyinstaller main.spec`
    2. The *ELF* will be built at `dist/matching_system`
